# config-plane

Welcome to the documentation for `config-plane`.

## What is it?

`config-plane` is a flexible configuration management library for Python. It implements a snapshot/stage model for configuration persistence, allowing you to manage application configuration with the same rigor as you manage your code.

-   **Version Controlled**: Track history of changes.
-   **Transactional**: Operations are atomic.
-   **Backend Agnostic**: Switch between Memory, Git, or SQL backends without changing application code.
